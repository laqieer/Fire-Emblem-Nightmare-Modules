How to use
----------

Load the corresponding chapter unit file (eg. bmap001.cmb) in the Nightmare program after LZ77-11 decompressing it (this can be done using the BATCH-LZ77 program for instance).

The chapter unit files can be found in the /dispos folder after extracting all the files from the ROM.
